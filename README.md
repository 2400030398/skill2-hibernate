# Skill-2 Hibernate CRUD Operations

## Description
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Hibernate ORM with MySQL database.

## Technologies Used
- Java
- Hibernate
- Maven
- MySQL
- Eclipse (STS)
- GitHub

## Project Structure
- Entity class: Product
- Utility class: HibernateUtil
- Main class: MainApp
- Configuration file: hibernate.cfg.xml

## CRUD Operations
- Insert product records
- Retrieve product by ID
- Update product price
- Delete product by ID
